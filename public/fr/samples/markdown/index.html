#

L’élément blockquote représente du contenu cité d’une autre source, optionnellement avec une citation qui doit être dans un élément footer ou cite, et optionnellement avec des modifications en ligne telles que des annotations et des abréviations.

Images
#

En plus du Shortcode Figure, vous pouvez définir des images avec la syntaxe Markdown suivante :

![Texte alternatif](image.jpg "Légende de l'image")

Vous pouvez également utiliser les ‘attributs Markdown’ de Hugo pour ajouter des attributs personnalisés à l’élément HTML créé. Par exemple, pour définir la largeur d’une image à 50% :

![Texte alternatif](image.jpg "Légende de l'image")
{style="width:50%;"}

Notez que la fonctionnalité ‘attributs Markdown’ nécessite une configuration du moteur de rendu Goldmark. Consultez la documentation Hugo pour plus d’informations.
