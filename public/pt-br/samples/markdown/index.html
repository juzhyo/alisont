O elemento blockquote representa conteúdo citado de outra fonte, opcionalmente com uma citação que deve estar dentro de um elemento footer ou cite, e opcionalmente com alterações em linha como anotações e abreviações.

Imagens
#

Além do Shortcode Figure, você pode definir imagens usando a seguinte sintaxe Markdown:

![Texto alternativo](image.jpg "Legenda da imagem")

Você também pode aproveitar os ‘atributos Markdown’ do Hugo para adicionar atributos personalizados ao elemento HTML criado. Por exemplo, para definir a largura de uma imagem para 50%:

![Texto alternativo](image.jpg "Legenda da imagem")
{style="width:50%;"}

Note que o recurso de ‘atributos Markdown’ requer configuração do renderizador Goldmark. Consulte a documentação do Hugo para mais informações.
